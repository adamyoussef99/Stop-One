In this website, we are using different technologis to implement the pages and store data.

front end:
HTML/CSS: used to create page layouts and style elements Javascript: used to show and reveal elements based on instance Angular 12: puts every component together to create our website’s pages Pug.js: used as our templating engine PHP: Importing API’s and handling the monitor page

back end:
Node.js: Used to launch Express.js web app Express.js: Creates web server to run through Node.js, also used to pull data from DB depending on client requests MongoDB: NoSQL database for user accounts, admin accounts, and products Mongoose: Middleware for MongoDB to connect our database and Node.js
